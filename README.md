# Almac and sons vehicle repairs Website

## Project Overview

This website was created as **Project 1** for the **Code Institute Diploma in Web Application Development**. The aim of the project is to design and build a static, responsive website using **HTML5 and CSS3**, demonstrating fundamental web development skills.

The website represents a **family-run local garage** offering MOT testing, vehicle servicing, and general repairs. It is designed to be clear, accessible, and easy to navigate for users of all ages.

---

## User Experience (UX)

### Target Audience

* Local vehicle owners
* Families looking for a trustworthy garage
* Customers needing MOTs, servicing, or repairs

### User Goals

* Easily understand what services are offered
* Learn about the garage and its values
* Find contact details quickly
* Book an appointment with minimal effort

### Business Goals

* Present a professional and trustworthy image
* Clearly advertise services
* Encourage customers to make contact or bookings

---

## Design

### Colour Scheme

* Dark grey navigation bar for contrast and professionalism
* Light background for readability
* White text on dark backgrounds for accessibility

### Typography

* Arial, sans-serif
* Chosen for clarity, familiarity, and accessibility

### Layout

* Consistent navigation across all pages
* Mobile-first responsive design
* Clear separation of content using sections

---

## Features

### Existing Features

* **Responsive navigation bar** visible on all pages
* **About page** explaining the business and its values
* **Services page** listing all garage services
* **Booking form** with required fields and dropdown selection
* **Contact information** including phone, email, and address

### Future Features

* Online booking system with date/time selection
* Google Maps integration
* Customer testimonials
* Image gallery of the garage
